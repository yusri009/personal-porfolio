# Personal Portfolio Website

A modern, responsive personal portfolio website built with React, Vite, and Tailwind CSS. This portfolio showcases my skills, projects, and provides a way for visitors to contact me directly through an integrated contact form.

## 🌐 Live Demo

The portfolio is deployed on Firebase Hosting and can be accessed at: yusriahamed.engineer

## 🛠️ Technologies Used

- **Frontend**: React 19, Vite
- **Styling**: Tailwind CSS 4.1
- **Routing**: React Router DOM
- **Icons**: React Icons
- **Email Service**: EmailJS
- **Build Tool**: Vite
- **Deployment**: Firebase Hosting
- **CI/CD**: GitHub Actions

## ✨ Features

- **Responsive Design**: Fully responsive layout that works on all devices
- **Single Page Application**: Smooth navigation with React Router
- **Interactive Contact Form**: Direct email functionality using EmailJS
- **Project Showcase**: Detailed project descriptions with embedded videos
- **Skills Display**: Visual representation of technical skills with icons
- **Social Media Integration**: Links to LinkedIn, GitHub, Facebook, and Instagram
- **Modern UI**: Clean, professional design with custom fonts and color scheme
